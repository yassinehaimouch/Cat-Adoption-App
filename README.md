## Run local

Clone the project

```bash
  git clone https://github.com/yassinehaimouch/Cat-Adoption-App.git
```

Enter the project directory 

```bash
  cd Cat-Adoption-App
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
